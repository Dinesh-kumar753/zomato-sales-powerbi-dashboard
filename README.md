# 🍽️ Zomato Sales Analysis Dashboard (Power BI)

## 📌 Overview

This project is an interactive Power BI dashboard that analyzes Zomato sales data to provide insights into outlet performance, item categories, sales trends, and customer ratings.

---

## 🎯 Objective

* Analyze overall sales performance
* Evaluate outlet-wise performance
* Understand item category trends
* Track customer ratings and satisfaction

---

## 📊 Key Features

* Total Sales, Average Sales, Total Items, Average Rating KPIs
* Outlet establishment trend over years
* Item type-wise sales distribution
* Outlet size and location analysis
* Fat content analysis (Low Fat vs Regular)
* Outlet tier comparison (Tier 1, 2, 3)
* Customer rating insights

---

## 📈 Key Insights

* Tier 3 outlets contribute highest item sales
* Regular fat products dominate sales (~65%)
* Medium-sized outlets generate the highest revenue
* Sales peaked around 2018 and stabilized afterward
* Fruits & Snacks categories are top-performing
